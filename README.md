# yunle-template-gulp

开发过程中，你用得最多的会是`npm run dev`，但是这里还有很多其它的处理：


|`npm run <script>`|用途|
|------------------|-----------|
|`dev`|开发环境服务。|
|`build`|生产环境服务打包服务。|

## 程序目录

```
.
├── .tmp                     # 开发临时文件
├── dist                     # 打包后文件
├── src                      # 应用源文件
│   ├── less                 # less源文件
│   ├── images               # images源文件
│   ├── js                   # js源文件
├── test                     # 单元测试
```
